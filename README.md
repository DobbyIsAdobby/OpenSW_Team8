<h3>🕹️ 프로젝트 내용(게임) 소개 </h3>
<h4> <게임 플레이 최종 목표></h4> 
지뢰 찾기를 통해 타일을 삭제해서, 타일 뒤에 숨겨진 이미지가 무엇인지 맞추는 게임입니다.
<h4><규칙></h4>
타일을 하나 삭제 +1 point<br>

이미지 정답 시 게임 CLEAR-!
<h4><조작></h4>
⬅️ 좌클릭: 타일 클릭<br>
➡️ 우클릭: 큐브 회전<br>
↕️ 스크롤: 줌인/아웃<br>
<h4><아이템 기능></h4>
게임 내에 한 번씩만 구매가능<br>
 &nbsp; 🔍돋보기 ( -200point ) <br> 3X3 타일을 공개한다. 대신 아이템 비용을 매우 높게 설정하여 최고 기록 경신은 사실상 불가능하게 만든다.<br>
 &nbsp; 🛡️3회 무적 ( -300point ) <br> 일정 횟수 타일을 해제할때 지뢰가 나와도 이미지가 훼손되지 않는다. 대신 이것도 아이템 비용을 매우 높게 설정한다.
