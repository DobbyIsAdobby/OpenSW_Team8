<h2>📖 GAME STORY</h2>
&nbsp; 수많은 역사의 파편들이 묻혀 있는 이 땅, 당신은 한국 문화유산 복원 및 보존을 목표로 하는 비밀 고고학 연구팀의 막내이자 유능한 신입 학자입니다.<br>
 &nbsp; 최근, 당신의 팀은 전설로만 전해져 오던 고대 기록물 ‘얼의 장(章)'을 해독하는 데 성공했습니다. 이 기록물에는 일제강점기와 한국전쟁 등 격변의 시기를 거치며 유실되거나 파괴된 국보급 문화유산의 잔상(殘像) 정보가 담겨 있었습니다.<br>
 &nbsp; 이 유산들은 시간이 흐르며 고대 방어 시스템과 알 수 없는 에너지장에 의해 ‘보호 큐브’ 속에 봉인된 상태이며, 이 큐브 속에는 ‘발굴 방해 트랩(지뢰)’가 설치되어 있어 무턱대고 발굴할 수 없습니다.<br>

<h3>🕹️ 프로젝트 내용(게임) 소개 </h3>
<h4> <게임 플레이 최종 목표></h4> 
지뢰 찾기를 통해 타일을 삭제해서, 타일 뒤에 숨겨진 이미지가 무엇인지 맞추는 게임입니다.

신뢰도 게이지가 있는데, 지뢰가 터지면 이미지가 훼손된다. 

<h4><규칙></h4>
타일을 하나 삭제 +1 point<br>

이미지 정답 시 게임 CLEAR-!
<h4><조작></h4>
⬅️ 좌클릭: 타일 클릭<br>
➡️ 우클릭: 큐브 회전<br>
↕️ 스크롤: 줌인/아웃<br>
<h4><아이템 기능></h4>
게임 내에 한 번씩만 구매가능<br>
 &nbsp; 🔍돋보기 ( -200point ) <br> 3X3 타일을 공개한다. 대신 아이템 비용을 매우 높게 설정하여 최고 기록 경신은 사실상 불가능하게 만든다.<br>
 &nbsp; 🛡️3회 무적 ( -300point ) <br> 일정 횟수 타일을 해제할때 지뢰가 나와도 이미지가 훼손되지 않는다. 대신 이것도 아이템 비용을 매우 높게 설정한다.
